HOW TO START : RUN THE access_location.py FOUND IN THE ZIP
---------------------------------------------------------------------

FIRSTLY,GO TO NEW SIGN IN AND CREATE AN ACCOUNT FOR ACCESSING THE ADMIN PART.
THEN ADD DATA'S OF BOOKS AND STUDENTS IN ADMIN SECTION.

---------------------------------------------------------------------

TASK DONE BY EACH FUCTION->
___________________________________________________________________________________________________

FUNCTIONS                                        TASK DONE
---------					 ---------
new_signin					for new admin sign in

checkemailandphone				check valid and phone no.

checkpas					check the criteria for valid password
------------------------------------------------------------------------------
login						admin login

insert_studentdata				to add student data to database

insert_bookdata					to add book data to database

delete_studentdata				to delete student data from database

delete_bookdata					to delete book data from database

whattodo					to initiate various functions
------------------------------------------------------------------------------
issue_func					to issue book to the student

return_func					to return the book issued by the student

books_by_student				lists of books issued by a student

students_fora_book				list of students issued a particular book

choice						asks librarian to perform certain actions

HOW TO ACCESS A FEATURE->
________________________________________________________________________________________________

-----------------------------------------------------
ADD A STUDENT ACC.:
	1.press [1] for admin login IN THE MAIN MENU
	2.perform the login using id and password
	3.press [2] for modifying a STUDENT INFORMATION
	4.press [1] for adding a STUDENT INFORMATION
-----------------------------------------------------
DELETE A STUDENT ACC.:
	1.press [1] for admin login IN THE MAIN MENU
	2.perform the login using id and password
	3.press [2] for modifying a STUDENT INFORMATION
	4.press [2] for deleting a STUDENT INFORMATION
-----------------------------------------------------
SHOW ALL STUDENT ACC.:
	1.press [1] for admin login IN THE MAIN MENU
	2.perform the login using id and password
	3.press [4] for showing all student acc.
-----------------------------------------------------
ADD A BOOK:
	1.press [1] for admin login IN THE MAIN MENU
	2.perform the login using id and password
	3.press [1] for modifying a BOOK INFORMATION
	4.press [1] for adding a BOOK INFORMATION
-----------------------------------------------------
DELETE A BOOK:
	1.press [1] for admin login IN THE MAIN MENU
	2.perform the login using id and password
	3.press [1] for modifying a BOOK INFORMATION
	4.press [2] for deleting a BOOK INFORMATION
-----------------------------------------------------
SHOW ALL BOOKS IN LIBRARY:
	1.press [1] for admin login IN THE MAIN MENU
	2.perform the login using id and password
	3.press [3] for the LISTS OF THE BOOK
-----------------------------------------------------
ISSUE A BOOK TO STUDENTS:
	1.press [2] for LIBRARIAN IN THE MAIN MENU
	2.press [1] for ISSUE BOOK
-----------------------------------------------------
RETURN A BOOK FROM STUDENTS:
	1.press [2] for LIBRARIAN IN THE MAIN MENU
	3.press [2] for RETURN A BOOK
-----------------------------------------------------
SHOWING THE LIST OF BOOKS AVAILABLE :
	1.press [2] for LIBRARIAN IN THE MAIN MENU
	2.press [3] for showing LIST OF BOOKS AVAILAVBLE IN LIBRARY.
-----------------------------------------------------
SHOWING A LIST OF BOOKS ISSUED BY A STUDENT:
	1.press [2] for LIBRARIAN IN THE MAIN MENU
	2.press [4] for showing lists of books issued by a student
-----------------------------------------------------
SHOWING A LIST OF STUDENTS ISSUE A PARTICULAR BOOK:
	1.press [2] for LIBRARIAN IN THE MAIN MENU
	2.press [5] for showing lists of STUDENTS issued A PARTICULAR BOOK.
-----------------------------------------------------

USE DATABASE FOR:
_______________________________________________________________________________

-----------------------------------------------------
admintolist			holds data's of the admin accounts

booktolist			holds data's of the books available in library

studenttolist			holds data's of the students registred for library

recordtolist			maintains the data's of students issued book(s)
